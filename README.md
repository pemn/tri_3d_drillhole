# tri_3d_drillhole
Create a 3d textured mesh from drillhole header, survey and any image
## features
 - Export result as gtlf binary (.glb) which is supported by Windows 3D Viewer
 - Export result a vtk mesh (.vtk)
 - Built in renderer using pyvista
